What is Cw3racl?
----------------

Cw3racl is common web3 rpc access client by lisp; Applicable to all evm compatible blockchains.


How to Install?
----------------
    apt-get install libcurl4-gnutls-dev
    pushd ./DependLibs
    bash CreateLib.sh
